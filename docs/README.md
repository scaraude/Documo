## Tech Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS v4, Shadcn UI
- **Backend**: tRPC, Prisma ORM
- **Database**: PostgreSQL
- **Testing**: Vitest (unit/integration), Playwright (E2E)
- **Linting/Formatting**: Biome
- **Package Manager**: Yarn
